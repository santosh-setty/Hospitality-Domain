# 🚀 Power BI Project – Hospitality Domain Performance Dashboard

I’m excited to share my **Power BI project** where I analyzed hotel booking data to understand revenue performance, room utilization, and booking trends across multiple cities and properties.

---

# 📊 Project Overview

This dashboard provides a complete view of hotel performance by analyzing bookings, revenue, occupancy, and cancellations. It helps stakeholders quickly understand demand patterns and operational efficiency across different properties.

---

# 📂 Dataset Used

The project is built using a **Star Schema Data Model** with the following tables:

| Table Name | Description |
|------------|-------------|
| dim_date | Date hierarchy and week analysis |
| dim_hotels | Property details, category, and city |
| dim_rooms | Room classification |
| fact_bookings | Booking level data and revenue |
| fact_aggregated_bookings | Aggregated booking capacity and successful bookings |
